API ENDPOINTS CONSUMED
================================================================================

1. GET /referrals/stats
   • Returns: ReferralStats (code, tier, counts, progress)
   • Called: On ReferralDashboardView appear, ProfileView appear

2. POST /referrals/apply
   • Body: ApplyReferralCodeRequest { code, source }
   • Called: ReferralCodeEntryView "Apply Code" button

3. POST /referrals/track-share
   • Body: ShareTrackingRequest { platform }
   • Called: After native share sheet dismissal
   • Platforms: imessage, email, whatsapp, instagram, link, other
