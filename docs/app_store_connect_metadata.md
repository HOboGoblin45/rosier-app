### Keyword Strategy Rationale

- **Primary Keywords:** niche fashion, fashion discovery, designer brands (high search volume, low competition)
- **Secondary Keywords:** style DNA, fashion app (product differentiation)
- **Long-tail Keywords:** shop curated, swipe fashion (low competition, specific intent)
- **Competitor-adjacent:** micro-influencer (target audience identifier)

**Avoid These:**
- Generic terms like "shopping," "fashion," "app" (too broad)
- Competitor names directly (violates guidelines)
- Misleading terms not in app (e.g., "sale alerts" if not in v1.0)

### 6.4 Important Privacy Notes

- **Affiliate Tracking:** When users click affiliate links, the affiliate network (Rakuten, Impact, etc.) may set their own tracking cookies. This is disclosed in the Privacy Policy and Terms of Service.

### 12.2 If Rejected

1. **Read rejection reason carefully** in App Store Connect
2. **Common rejection reasons and fixes:**
   - Missing privacy policy → Add valid URL
   - Crash on startup → Test locally, fix, resubmit
   - Misleading metadata → Update description/screenshots
   - Missing demo account → Provide valid test credentials
   - Affiliate links not disclosed → Add to Terms of Service

3. **Resubmission process:**
   - Make fixes locally
   - Increment build number (CURRENT_PROJECT_VERSION = 2)
   - Build and archive new version
   - Upload new build to App Store Connect
   - Update "What's New" text explaining fix
   - Submit new build for review
